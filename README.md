## 개요
- 프로젝트 이름: fasion style 커뮤니티 
- 프로젝트 목적 : 
    -   누구나 쉽게 이용 가능한 패션 커뮤니티 공유 사이트를 개설
    -   조회수 등을 바탕으로 얻은 데이터를 통해 소비자들의 니즈 파악
- 프로젝트 지속기간: 2023.9.1 - 2013.9.8
- 개발 엔진 및 언어: python(flask), html, mysql
- 멤버: 김상훈, 윤규희, 윤석현, 최인혁, 전경원
